## Sample

See also:  [@shikijs/transformers | Shiki](https://shiki.style/packages/transformers)

### Highlight

````markdown
```ts
console.log('Not highlighted')
console.log('Highlighted') // [!code highlight]
console.log('Not highlighted')
```
````

```ts
console.log('Not highlighted')
console.log('Highlighted') // [!code highlight]
console.log('Not highlighted')
```

### Word Highlight

````markdown
```ts
// [!code word:Hello]
const message = 'Hello World'
console.log(message) // prints Hello World
```
````

```ts
// [!code word:Hello]
const message = 'Hello World'
console.log(message) // prints Hello World
```

### Focus
